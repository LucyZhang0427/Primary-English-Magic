# 🌟 Lucy 老师的英语魔法屋

欢迎来到我的线上教室！这里专门为小学生提供有趣的英语学习资源。

## 📅 本周课程主题：Animals (动物)
本周我们将学习如何描述自己喜欢的动物。

### 1. 核心词汇 (Vocabulary)
*   **Elephant** (大象) - *It has a long nose.*
*   **Monkey** (猴子) - *It likes bananas.*
*   **Panda** (熊猫) - *It is black and white.*

### 2. 趣味任务 (Task)
请小朋友们画一只你最喜欢的动物，并尝试用英文说出它的名字，录制小视频发送到群里哦！
